# CalcSage 🧮

CalcSage is a sleek, responsive web-based calculator designed for quick and accurate calculations. Built with modern frontend technologies, it delivers a clean UI and smooth user experience.

## 🚀 Features
- Simple and intuitive interface
- Supports basic arithmetic operations
- Fully responsive design
- Fast and lightweight

## 🛠️ Built With
- HTML, CSS, JavaScript

## 📌 Usage
1. Clone the repo:
   ```bash
   git clone https://github.com/lcc33/calcsage.git
   ```
2. Open `index.html` in your browser.
